# はんなりPython / Dash Hands On 20200319

### ハンズオン用リポジトリ

```
git clone https://github.com/hannari-python/dash_hands_on.git
cd dash_hands_on
pip install -r requirements.txt
もしくは
pip install dash plotly pandas 
```

#### ファイルの実行

```
$ python first.py
```

``http://127.0.0.1:8050/`` にアクセスすると、アプリケーションが見れます。
